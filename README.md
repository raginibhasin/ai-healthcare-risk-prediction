# AI-Powered Healthcare Risk Prediction System

This project is a Flask-based healthcare web application that predicts patient health risk using a machine learning model.

Features:
• Patient Registration
• AI Risk Prediction
• Patient Records Management
• Dashboard Analytics
• CSV Export
• Responsive UI

Tech Stack:
Python
Flask
Scikit-learn
SQLite
HTML/CSS
Bootstrap

Future Improvements:
• Authentication system
• Cybersecurity enhancements
• Bulk data upload
